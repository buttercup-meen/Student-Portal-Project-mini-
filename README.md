# Student Information Portal 🎓

### **Project Overview**
This is a web-based student management system developed for the Web Engineering Lab. It allows students to register, view their profiles, and manage course enrollments.

### **Features**
* **Registration:** Form validation for empty fields, password matching, and age verification.
* **Dashboard:** Displays user-specific data (Name, Email, Age) fetched from **LocalStorage**.
* **Course Management:** Interactive checkbox selection for academic courses.
* **Responsive Design:** Built using **Bootstrap 5** for a centered, mobile-friendly layout.

### **Technologies Used**
* **HTML5:** Structure
* **Bootstrap 5:** Styling and Layout
* **JavaScript:** DOM Manipulation, Events, and Web Storage API
